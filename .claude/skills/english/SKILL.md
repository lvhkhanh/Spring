# English Communication and Writing Skill

## Overview

This skill helps with English writing and editing tasks, including grammar correction, style improvements, tone adjustment, clarity enhancement, and concise summarization.

## Key Capabilities

### Grammar and Spelling
- Correct punctuation, capitalization, and syntax
- Fix common spelling errors and typos
- Resolve subject-verb agreement, tense consistency, and sentence fragments

### Style and Tone
- Rewrite text for formal, informal, professional, or conversational tone
- Improve readability and coherence
- Recommend or apply concise wording and flow

### Structure and Organization
- Create outlines and sections from rough notes
- Generate clear introductions, body sections, and conclusions
- Improve paragraph structure and transition usage

### Summarization and Expansion
- Summarize long text into short and accurate points
- Expand key ideas into fuller paragraphs while preserving meaning
- Create abstracts, bullets, or executive summaries

## Usage Examples

### Rewrite for Clarity
```
Rewrite this paragraph to be clear and concise:
- "The project is going to be done, maybe, if we coordinate the teams properly and the weather holds up, we might be able to get it done by Friday."
```

### Formal Tone Conversion
```
Convert to formal business email style:
- "Hey team, just checking in on the status of the release. It would be great if we can finish soon."
```

### Grammar Correction
```
Fix grammar and spelling:
- "She have went to the store and buys two apple's."
```

### Summary
```
Summarize this article in 3 bullet points:
- [Input text block]
```

## Common Patterns

### Peer Review Edits
- Identify awkward phrasing and replace with crisp, natural expressions
- Maintain original meaning while improving flow

### Conversion Tasks
- Translate between formal/informal, short/long, simple/advanced styles
- Preserve context when changing tone

## Best Practices

- Ask for the intended audience and purpose when possible
- Provide side-by-side comparisons of original vs revised text
- Keep sentences shorter for improved readability
- Use active voice when appropriate, passive voice where necessary for formality
- Avoid changing factual content while editing grammar

## Troubleshooting

### Common Issues
- Input text too long: split into smaller chunks
- Ambiguous intent: ask clarifying questions about target tone or audience
- Mixed language: identify primary language and confirm if translation is intended

### Validation Steps
1. Re-read the output for meaning preservation
2. Compare key points with original text for accuracy
3. Check for new grammar errors introduced in revision

## Integration Points

- **Documentation**: improve manuals, readmes, technical guides
- **Email**: draft professional communications
- **Reports**: summarize meeting notes and status updates
- **Learning**: explain grammar rules with examples
- **Content**: generate blog posts, articles, and social media copy
